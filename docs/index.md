# SQLitePlus Enhanced

SQLitePlus Enhanced reúne en un mismo paquete todo lo necesario para trabajar cómodamente con bases SQLite. Puedes montar una API moderna o limitarte a gestionar la base desde la consola, según lo que necesites en cada momento.

## Qué ofrece

- **Operaciones asincrónicas** que cuidan el acceso concurrente a varias bases.
- **Inicio de sesión con JWT** usando un sencillo archivo JSON de usuarios.
- **Compatibilidad con SQLCipher**, sin complicaciones extra para quien no lo necesite.
- **Copias y exportaciones guiadas** para obtener CSVs o respaldos fechados con un comando.
- **CLI `sqliteplus`** con mensajes claros que acompañan cada paso.
