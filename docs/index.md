# SQLitePlus Enhanced

SQLitePlus Enhanced es una librería avanzada de gestión de bases de datos SQLite que incluye:

- 🔐 Autenticación JWT
- 🔑 Cifrado con SQLCipher
- ⚡ Caché con Redis
- 🚀 API REST asíncrona con FastAPI
- 🔧 CLI para operaciones locales
- 📦 Compatible con PyPI
