# SQLitePlus Enhanced

SQLitePlus Enhanced brings together everything you need to work comfortably with SQLite databases in a single package. You can set up a modern API or just manage the database from the console, depending on what you need at the moment.

## What it offers

- **Asynchronous operations** that handle concurrent access to multiple databases carefully.
- **JWT Login** using a simple JSON users file.
- **SQLCipher compatibility**, without extra complications for those who don't need it.
- **Guided backups and exports** to obtain CSVs or dated backups with a single command.
- **`sqliteplus` CLI** with clear messages accompanying each step.

Explore the side menu to follow the installation, configuration, and step-by-step usage examples.
